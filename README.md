# WonderGolf
Top down daily miniature golf game

## 🚀 Getting the Live Site Up on GitHub Pages

To play WonderGolf at **https://jsr1151.github.io/wondergolf/**, follow these two steps:

### Step 1 — Enable GitHub Pages (one-time setup)
1. Go to **https://github.com/jsr1151/wondergolf/settings/pages**
2. Under **"Build and deployment"**, set the **Source** dropdown to **"GitHub Actions"**
3. Click **Save**

### Step 2 — Merge this PR
1. Go to **https://github.com/jsr1151/wondergolf/pull/2**
2. Click **"Ready for review"** (to un-draft the PR), then **"Merge pull request"**
3. The **Deploy to GitHub Pages** workflow will run automatically

After merging, the deployment takes about 30–60 seconds. You can watch it at:
**https://github.com/jsr1151/wondergolf/actions**

Once it's green, the game is live at **https://jsr1151.github.io/wondergolf/**

## 🏌️ About the Game

A browser-based, top-down daily miniature golf game — no install required, works on desktop and mobile.

- **Daily rounds** — 9 holes selected each day via seeded RNG (same course for everyone worldwide)
- **Putting** — click-and-drag slingshot mechanic with aim line and power bar
- **Hazards** — sand (slows ball), water (penalty + reset), slick zones (speeds ball up)
- **Progression** — XP, levels, and achievements saved in your browser
